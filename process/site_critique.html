Switzerland

https://www.myswitzerland.com/en-us/

Navigation

I really like the navigation of the website. I like the hover effects, with the color changes.I also like how the background of the navigation bar shows up. I also like how they have a search bar as there is a lot of information on the page.

Usability

There are 2 navigation bars, which can be confusing to some people. The main navigation bar each link has a drop down. The drop downs are really big and have a lot of information which could be hard to use. I do like the zoom in on hover feature, the ability to change languages, and book a hotel.

Typography and hierarchy

The Typography is pretty basic and boring. This website uses a typical modern font. The hierarchy is also pretty basic with the more important stuff being bolded. It is very clear what the important information is.

Overall style/look and feel

Although the website has some nice features, I do feel like there is too much going on. For example, the main picture on the home page, turns into an interactive map with a navigation bar. This was pretty confusing at first. I also just feel like the home page has too much information. However, I do like the colors, pictures, and Typography of this website
